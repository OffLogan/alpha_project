# User Flows

## Home: tasks

### Create a task
1. Write a name in the task name field.
2. Optionally add a description.
3. Press `Add`.

### Edit a task
1. Double click a task in the task list.
2. Choose `Edit`.
3. Update the fields and confirm.

### Complete a task
1. Double click a task.
2. Choose `Complete`.

### Delete a task
1. Double click a task.
2. Choose `Delete`.
3. Confirm the action.

### Filter tasks
Use the task filter above the task list to switch between:
- `Pending`
- `Completed`

## Home: reminders

### Create a reminder
1. Write a name in the reminder name field.
2. Optionally add a description.
3. Enter a due date using `DD/MM/YY`.
4. Press `Add`.

### Edit a reminder
1. Double click a reminder in the reminder list.
2. Choose `Edit`.
3. Update the fields and confirm.

### Complete a reminder
1. Double click a reminder.
2. Choose `Complete`.

### Delete a reminder
1. Double click a reminder.
2. Choose `Delete`.
3. Confirm the action.

### Filter reminders
Use the reminder filter to switch between:
- `Pending`
- `Completed`
- `Overdue`
- `Due soon`

### Sort reminders
Use the reminder sort selector to order by:
- `Due soonest`
- `Due latest`

## Global search
1. Press `Search` in the home screen.
2. Enter a search term.
3. Review the result list.
4. Double click a result to open the relevant module or item.

The current search covers:
- tasks
- reminders
- notes
- schedule entries

## Notes

### Create a note
1. Select a folder if you want to create the note inside it.
2. Press `New note`.
3. The editor opens for the new note.

### Create a folder
1. Press `New folder`.
2. Enter the folder name.
3. Optionally enter a parent folder name.

### Rename a folder
1. Select a folder in the tree.
2. Press `Rename folder`.
3. Enter the new name.

### Move a note or folder
1. Select the item in the tree.
2. Press `Move item`.
3. Choose the destination folder or `Root`.

### Open and edit a note
1. Double click a note in the tree.
2. Edit the title and content in the editor.
3. Save the note.

### Export a note
1. Select a note.
2. Press `Download TXT` or `Download PDF`.

## Schedule

### Add a schedule block
1. Choose the day.
2. Enter the time.
3. Enter the subject.
4. Optionally enter the location.
5. Press `Add block`.

### Edit a schedule block
Double click a table cell and edit it directly.

### Delete a schedule block
1. Select a row in the table.
2. Press `Delete selected`.

### Save the schedule
Press `Save schedule`.

### Export the schedule
Press `Export`.

### Import the schedule
Press `Import` and choose a JSON file.

### Clear the schedule
1. Press `Clear all`.
2. Confirm the action.

## Settings

### Change theme
1. Open `Settings`.
2. Toggle `Dark mode`.
3. Confirm with `Ok` to keep the change.
4. Use `Cancel` to restore the previous state.

### Change notification preference
1. Open `Settings`.
2. Toggle `Enable notifications`.
3. Confirm with `Ok`.

### View app information
`Settings` shows:
- current version
- local data path
