# KMN Space
A personal academic workspace for notes, tasks, reminders, and schedules.

Current version: 0.0.1.

# framework
The application has been developed in C++ using Qt as the main framework for building the graphical user interface, managing windows, and handling user interaction. Qt was chosen for its strong integration with C++ and for its ability to support the development of modular, scalable, and cross-platform desktop applications.
